ab-kurs-kayit

Akademik Bilişim Konferansı Öncesi kurs kayıt ve izleme araçları

- Django 1.7

- Python 2.7

- python-devel

- Postgresql 9.2

- postgresql-server

- postgresql-contrib

- django-settings-context-processor

- postgresql_psycopg2

-  django-ckeditor

- django-country

Apache ile kullanmak icin gereksinimler:

-- apache2

-- mod_wsgi

-- mod_ssl
